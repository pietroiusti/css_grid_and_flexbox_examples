janus-key |
C ⸱ Linux Input Event Interface
A key remapper for GNU/Linux that provides keys with a double function.
I am a daily Emacs user and using Emacs means a persistent usage of modifiers; not the best treatment for your hands. In order to improve ergonomics, I came up with the idea of turning normal keys into modifiers only when pressed down, keeping their original functionality when tapped. janus-key allows to assign any key two functions, one when tapped, and another when held down and used in combination with other keys. As far as I know, janus-key is the first piece of software of its kind that it's both graphical interface independent — besides X, it works on Wayland, and on the tty — and systemd independent. I first implemented this idea in python, but it was too good an opportunity to write my first “real” C program. I use it everyday!
